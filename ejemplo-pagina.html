Cambio de color en texto usando estilos.

Para cambiar el color de algun texto se utiliza la etiqueta style y el color deseado, asi el texto cambiara de color y podremos mejorar el diseño de la pagina.

Tambien podemos cambiar el tamaño de las letras usando la etiqueta de font size dejando asi el texto como uno desee.

Colocar imagenes o gif.

Para colocar una imagen o un gif se utiliza la etiqueta img le ponemos src para buscar la fuente de la imagen, esto se puede colocar con imagenes en el computador o imagenes online sacando la url de la imagen.
